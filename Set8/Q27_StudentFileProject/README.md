# Student File Handling in Java

## Description
This program demonstrates file handling in Java.
It creates a file, writes student records, and reads them.

## Features
- Create file (students.txt)
- Write student data
- Read and display records
- Exception handling using try-catch-finally

## How to Run
javac StudentFileDemo.java  
java StudentFileDemo
