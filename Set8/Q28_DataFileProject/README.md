# Data File Count Program in Java

## Description
This Java program reads a file named data.txt and counts:
- Total number of lines
- Total number of words
- Total number of characters (excluding spaces and newline)

## Features
- Uses FileReader and BufferedReader
- Handles FileNotFoundException and IOException
- Simple and easy to understand

## How to Run
javac DataFileCount.java  
java DataFileCount

## Sample Input (data.txt)
Hello world  
Java programming is fun

## Output
Total Lines: 2  
Total Words: 5  
Total Characters: (excluding spaces)
